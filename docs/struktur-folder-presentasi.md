# Struktur Folder Presentasi

Dokumen ini merangkum folder dan file penting yang akan ditampilkan saat presentasi.

- app/Http/Controllers = tempat controller berdasarkan role Admin, SPV, Teknisi
- app/Models = tempat model/entity Laravel
- database/migrations = struktur tabel database
- database/seeders = data awal dan demo data
- resources/views = tampilan halaman web
- routes/web.php = pengaturan URL dan akses role
- public = asset publik aplikasi
- storage = penyimpanan file upload lampiran
- .env = konfigurasi local database dan app key
- composer.json = dependency backend Laravel
- package.json = dependency frontend/Vite
